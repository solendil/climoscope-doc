# ⌨️ Keyboard shortcuts

> when applicable, and only on a computer

- `←` `→` left and right arrows for previous and next **day**
- `↓` `↑` down and up arrow for previous and next **month**
- `⤓` `⤒` page-down and page-up for previous and next **year**
- `e` next element (tx, tn, etc...) `shift` for previous
- `d` next "duration" (day, month or year) `shift` for previous
- `m` next mode (view, compare, etc...) `shift` for previous
- `r` reset, if needed
